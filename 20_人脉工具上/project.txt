项目的主要功能和架构：

### 1. 项目功能
这是一个基于微博大V信息的人脉工具，主要功能是：

- 输入一个关键词（如花的名称）
- 自动查找相关的微博大V
- 分析大V的信息
- 生成联络建议
### 2. 系统架构
项目采用前后端分离的架构：

1. 前端部分
```plaintext
/templates/index.html  - 网页界面
/static/css/style.css - 页面样式
 ```

2. 后端部分
- 核心模块：
  
  - app.py - Flask应用主入口
  - findbigV.py - 核心业务逻辑
- 功能模块：
```plaintext
/agents/
  - weibo_agent.py   # 微博用户查找代理

/tools/
  - search_tool.py   # 搜索工具
  - scraping_tool.py # 数据爬取工具
  - parsing_tool.py  # 数据解析工具
  - textgen_tool.py  # 文本生成工具
  - general_tool.py  # 通用工具
 ```

### 3. 工作流程
1. 查找阶段
   
   - 用户输入关键词
   - 使用 SerpAPI 搜索相关微博用户
   - 提取用户 UID
2. 数据获取阶段
   
   - 根据 UID 爬取用户信息
   - 过滤无关信息
   - 保留中文内容
3. 分析生成阶段
   
   - 生成用户简介
   - 提取特点和兴趣
   - 生成联络建议
4. 展示阶段
   
   - 格式化信息
   - 通过 Web 界面展示
   - 支持动态加载
### 4. 技术特点
- 使用 LangChain 框架
- 集成 GPT-3.5-turbo 模型
- 采用代理模式处理复杂任务
- 支持结构化数据解析
- 实现了异步加载和友好的用户界面
这个项目展示了如何将 AI 能力应用到实际的社交网络分析中，通过自动化的方式帮助用户建立人脉关系。
